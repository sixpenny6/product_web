# 02 - Stage 1: Project Initialization

执行阶段 1：项目初始化。

目标：

- 初始化 Next.js App Router + TypeScript + Tailwind 项目；
- 使用 `src` 目录；
- 使用 `@/*` 路径别名；
- 添加 ESLint；
- 创建基础页面：
  - `/`
  - `/products`
  - `/products/[slug]`
  - `/applications`
  - `/applications/[slug]`
  - `/downloads`
  - `/contact`

限制：

- 不接 CMS；
- 不做 3D；
- 不做真实产品数据；
- 不做在线支付；
- 不实现复杂业务逻辑。
