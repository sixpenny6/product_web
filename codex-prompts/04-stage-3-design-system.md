# 04 - Stage 3: Design System

执行阶段 3：设计系统。

请根据 `docs/tubular_motor_website_docs/03_UI视觉设计规范.md` 实现基础 UI 组件：

- Button
- Card
- Badge
- GlassPanel
- SectionHeader
- SpecBadge
- CTAGroup

要求：

1. TypeScript 类型完整；
2. 支持 `className`；
3. 支持 hover、focus-visible、disabled；
4. 深色工业科技风；
5. 电蓝主强调色，铜色作为少量点缀；
6. 不要引入复杂业务逻辑；
7. 不要使用真实产品参数。

完成后创建一个临时 UI preview section 或在首页展示这些组件，方便验收。
