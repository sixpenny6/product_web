# 内容素材清单：给工厂、销售、工程和设计师

## 1. 每个产品必须提供的信息

- 产品名称
- 型号
- 产品系列
- 应用场景
- 一句话卖点
- 详细描述
- 扭矩
- 速度
- 管径
- 电压
- 功率
- 电流
- 限位方式
- 控制方式
- 防护等级
- 噪音
- 工作温度
- 尺寸
- 重量
- 认证
- 适配附件
- 包装信息
- MOQ，可选
- 保修政策，可选

## 2. 每个产品必须提供的文件

- 产品高清图
- 尺寸图
- 安装说明
- 接线图
- 规格书
- 认证文件
- CAD / STEP 文件，用于 3D 建模
- 产品铭牌信息
- 包装图，可选

## 3. 拍摄建议

### 3.1 产品照片

- 正面
- 侧面
- 电机头部
- 线缆细节
- 支架细节
- 管径适配件
- 铭牌
- 安装在卷管中的状态
- 应用场景图

### 3.2 360 图片序列

如果暂时没有 3D 模型，可以拍 36 张或 72 张图。

要求：

- 固定相机
- 固定光源
- 白底或深灰底
- 产品每次旋转 5°或 10°
- 图片命名连续

示例：

```txt
series-92-motor-000.jpg
series-92-motor-010.jpg
series-92-motor-020.jpg
```

## 4. 3D 建模资料

给建模师：

- STEP / CAD
- 产品真实尺寸
- 产品材质说明
- 颜色
- Logo / 铭牌
- 哪些部件要能点击
- 哪些部件要做 exploded view，可选
- 哪些部件不能展示，涉及商业保密时需要简化

## 5. 文案资料

每个产品需要：

- Hero 标题
- 副标题
- 3-5 个核心卖点
- 应用描述
- 安装说明摘要
- FAQ 5-8 个
- 询盘引导语

## 6. FAQ 示例

- How do I choose the right tubular motor torque?
- Which tube diameter does this motor fit?
- Does it support remote control?
- Is it suitable for garage doors?
- What documents are available for installation?
- Can I request OEM or ODM customization?
- What is the difference between 92 Series and 92 Servo Series?

## 7. 素材命名规范

```txt
/product-series/model-name/
  images/
  docs/
  models/
  videos/
```

示例：

```txt
/series-92/tm92-standard/
  images/tm92-standard-hero.webp
  docs/tm92-standard-datasheet-en.pdf
  models/tm92-standard-v1.glb
```
