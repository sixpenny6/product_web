## 2. 首页结构

### 2.1 Hero 区

目标：第一屏就建立科技感和产品认知。

内容：

- 暗色背景
- 中间一个管状电机 3D 模型或高质量渲染图
- 主标题：Tubular Motor Systems for Shading, Shutters and Garage Doors
- 副标题：Precision drive solutions for residential, commercial and industrial openings.
- CTA 1：Explore Products
- CTA 2：Find the Right Motor
- 右侧或下方快速参数标签：
  - Small to Heavy-Duty
  - 59 / 92 Series
  - Servo Control Available
  - 360° Product View

### 2.2 产品系列入口

四张产品卡：

1. Small Tubular Motors
2. 59 Series Motors
3. 92 Series Motors
4. 92 Servo Motors

每张卡包含：

- 产品渲染图
- 典型应用
- 扭矩范围，待用户补充真实数据
- Learn More 按钮
- Compare 按钮

### 2.3 应用场景

用大图或 3D 场景卡片展示：

- 遮阳帘 / 窗帘
- 小卷帘门
- 车库门
- 大型卷帘门 / 工业门

每个场景都引导到对应产品。

### 2.4 技术优势

建议 4 个模块：

- Quiet Drive
- Durable Gear System
- Smart Control Ready
- Easy Installation

每个模块可用短动效展示，不要写长段文字。

### 2.5 360°展示引导

放一个交互预览区：

- 标题：Inspect Every Detail in 360°
- 展示一台 92 系列电机
- 可拖动旋转
- 热点：Motor Head / Tube Adapter / Cable / Limit System / Mounting Bracket

### 2.6 资料与询盘 CTA

底部放强 CTA：

- Need torque, tube size or control compatibility?
- Talk to our technical team.
- Get Quote

## 4. 产品详情页

### 4.1 页面骨架

1. Breadcrumb
2. Product Hero
3. 360°/3D Viewer
4. Quick Specs
5. Application Fit
6. Key Features
7. Technical Specifications
8. Dimensions & Installation
9. Wiring / Control Options
10. Downloads
11. Related Accessories
12. Related Products
13. FAQ
14. Inquiry CTA

### 4.2 Hero 区

左侧：

- 产品名
- 产品系列
- 一句话定位
- 适用场景标签
- Get Quote
- Download Datasheet

右侧：

- 产品 3D 模型
- 旋转提示：Drag to rotate
- 3D / Images / Dimensions 三个模式切换

### 4.3 360°查看器

功能：

- 鼠标拖拽旋转
- 触控旋转
- 滚轮缩放，可限制缩放范围
- 热点点击
- 自动旋转开关
- 重置视角
- 全屏查看
- 移动端轻量模式
- 低性能设备降级为 360 图片序列

热点建议：

- Motor Head
- Gearbox
- Limit Adjustment
- Power Cable
- Tube Adapter
- Mounting Bracket
- Servo Encoder，仅 92 伺服电机
- Control Interface，仅智能/伺服型号

### 4.4 规格区

规格表需要结构化，而不是图片。

字段建议：

- Model
- Series
- Application
- Rated Torque
- Rated Speed
- Tube Diameter
- Voltage
- Frequency
- Current
- Power
- Duty Cycle
- Noise Level
- Protection Rating
- Limit Type
- Control Type
- Operating Temperature
- Cable Length
- Certifications
- Weight
- Dimensions

### 4.5 下载区

每个产品至少提供：

- Datasheet PDF
- Installation Manual PDF
- Wiring Diagram PDF
- Certification PDF，可选
- CAD / STEP / GLB，可选
- Product Images Zip，可选
