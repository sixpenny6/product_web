# 前端功能模块与组件清单

## 1. 项目结构建议

```txt
src/
  app/
    [locale]/
      page.tsx
      products/
        page.tsx
        [slug]/
          page.tsx
      applications/
        page.tsx
        [slug]/
          page.tsx
      downloads/
        page.tsx
      contact/
        page.tsx
      search/
        page.tsx
  components/
    layout/
    navigation/
    product/
    viewer3d/
    forms/
    sections/
    ui/
  lib/
    api/
    cms/
    seo/
    validators/
    constants/
  styles/
  types/
```

如果 MVP 暂时不做多语言，可以去掉 `[locale]`，但建议从一开始保留国际化扩展位。

## 2. Layout 模块

### AppHeader

功能：

- Logo
- 顶部导航
- Mega Menu
- 搜索入口
- Get Quote CTA
- 移动端菜单

### MegaMenu

产品导航必须清楚：

- 按系列
- 按应用
- 按控制类型
- 快速入口：Compare / Downloads / Contact Sales

### Footer

包含：

- 产品系列
- 应用场景
- 下载
- 联系方式
- 社媒
- 备案/隐私政策

## 3. 首页模块

### HomeHero

- 3D/视频/渲染图背景
- 主标题
- 副标题
- 双 CTA
- 快速参数标签

### ProductSeriesGrid

- 四大系列入口
- 支持 hover 动效
- 支持移动端横滑

### ApplicationShowcase

- 场景卡片
- 推荐产品
- Learn More

### TechnologyHighlights

- 安静
- 耐用
- 智能控制
- 易安装

### HomeViewerPreview

- 简化版 360°查看器
- 引导用户进入产品详情

## 4. 产品模块

### ProductFilterSidebar

字段：

- 系列
- 应用
- 扭矩
- 管径
- 电压
- 控制方式
- 防护等级
- 是否伺服

### ProductCard

字段：

- 图
- 名称
- 系列
- 应用
- 核心规格
- View Details
- Compare

### ProductCompareDrawer

功能：

- 添加/移除产品
- 打开对比页或弹窗
- 移动端底部固定

### ProductSpecsTable

支持：

- 分组规格
- 高亮关键参数
- 空值显示 `—`
- 移动端卡片化

### ProductDownloads

字段：

- 文件名
- 类型
- 语言
- 版本
- 大小
- 下载按钮

## 5. 3D 模块

### ProductViewer3D

见《04_360产品查看器技术方案.md》。

### ProductHotspotDrawer

字段：

- 热点标题
- 说明
- 关联参数
- 图片
- CTA

### ModelFallback

当 WebGL 或模型失败时显示：

- 静态产品图
- 360 图片序列
- 提示“3D view unavailable on this device”

## 6. 表单模块

### InquiryForm

字段：

- 姓名
- 公司
- 邮箱
- 电话
- 国家/地区
- 感兴趣产品
- 应用场景
- 预计数量
- 留言
- 附件，可选
- 隐私同意

校验：

- 邮箱必填
- 姓名必填
- 留言或产品兴趣至少一个
- 附件大小限制

### NewsletterForm，可选

如果没有持续内容运营，MVP 可不做。

## 7. 搜索模块

### SiteSearch

搜索范围：

- 产品名
- 型号
- 分类
- 应用
- 技术文档
- FAQ

MVP 可以使用数据库 LIKE 或简单全文搜索。
后续可接 Meilisearch / Algolia。

## 8. SEO 模块

### generateMetadata

每个页面需要：

- title
- description
- Open Graph
- canonical
- robots
- structured data

产品页建议使用 Product Schema，但 B2B 产品没有公开价格时，注意不要伪造价格。

## 9. 动效模块

### MotionSection

统一管理进入视口淡入、上移动效。

### ScrollFeatureSection

用于首页/技术页分段叙事。

注意：不要滚动劫持，不要影响用户找按钮。

## 10. 前端验收标准

- 导航清楚，产品入口不超过 2 次点击。
- 首页首屏能看懂公司卖什么。
- 产品详情页首屏能看到产品名、产品图、CTA。
- 3D 模块可用但不阻塞内容。
- 移动端底部有询盘 CTA。
- 产品筛选结果更新流畅。
- 表单提交有 loading、success、error 状态。
- 所有按钮 hover/focus 状态完整。
- 键盘可操作主要功能。
- 页面在无 JS 或弱网情况下仍显示核心内容。
