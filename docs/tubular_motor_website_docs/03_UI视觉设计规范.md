# UI 视觉设计规范

## 1. 设计方向

整体风格：高级工业科技感。

不是消费电子的纯炫酷，也不是传统机械厂的老式目录站。应该介于 DJI、Dyson、Razer 和 Dell 企业级硬件页之间：

- DJI：场景化、分段叙事、产品影像高级。
- Dyson：干净、质感强、配置/可视化清楚。
- Razer：暗色高对比、科技感强、参数视觉化。
- Dell PowerEdge：B2B 信息结构清楚，3D/AR 入口明确。

## 2. 颜色系统

### 主色

- Background Dark: #05070A
- Surface Dark: #0B0F14
- Card Dark: #111820
- Text Primary: #F4F7FA
- Text Secondary: #A8B3C2
- Border Subtle: rgba(255,255,255,0.12)

### 强调色

建议选一种主强调色，不要混太多。

方案 A：工业电蓝

- Accent Blue: #2F7DFF
- Accent Cyan: #00D5FF

方案 B：高端金属铜

- Accent Copper: #C8783A
- Accent Amber: #FFB45C

方案 C：Razer 风绿色，不建议大面积用

- Accent Green: #44D62C

最终建议：以电蓝为主，铜色用于局部结构热点和工业质感。

## 4. 按钮规范

### Primary Button

用于主要转化。

- 文案：Get Quote / Explore Products / Find Your Motor
- 样式：实色电蓝或铜色
- 圆角：12px
- 高度：44-52px
- Hover：轻微发光 + 上移 1px

### Secondary Button

用于下载、了解更多。

- 透明背景
- 1px 边框
- Hover：背景变亮

### Tertiary Button

用于小操作。

- 文本按钮
- 带箭头
- 用于 Learn More / View Specs

## 5. 导航与引导

### 顶部导航

- 桌面端使用 Mega Menu。
- 移动端使用全屏抽屉导航。
- “Products”和“Get Quote”必须始终明显。

### 页面内锚点导航

产品详情页建议有 sticky anchor nav：

- Overview
- 360 View
- Specs
- Dimensions
- Downloads
- FAQ

### CTA 放置规则

每个长页面至少出现 3 次明确 CTA：

1. Hero 区
2. 规格/下载区附近
3. 页面底部

产品页右侧可有 sticky inquiry bar：

- Get Quote
- Download Datasheet
- Compare

## 6. 卡片规范

### 产品卡片

必须包含：

- 产品图
- 产品系列
- 产品名称
- 典型应用
- 核心参数
- View Details
- Compare

不要只放大图和一句宣传语。

### 参数卡片

用数字强化：

- Torque
- Speed
- Tube Diameter
- Control Type

例：

```
Torque
XX Nm
For garage door systems
```

具体数值等待真实产品数据补充。

## 7. 3D 查看器 UI

### 查看器顶部

- 产品名称
- 当前模式：3D / Dimensions / Exploded，可选
- Reset View
- Fullscreen

### 查看器内部提示

首次进入显示：

- Drag to rotate
- Scroll to zoom
- Click hotspots for details

移动端显示：

- Swipe to rotate
- Pinch to zoom

### 热点样式

- 小圆点
- 外圈呼吸动画
- 点击后打开右侧信息面板
- 面板内容必须短、清楚，并带 Learn More / View Specs

## 8. 动效原则

动效要服务理解，不要干扰找按钮。

可以用：

- 产品缓慢自动旋转
- 进入视口时淡入
- 参数数字滚动
- 背景网格或光线流动
- 热点呼吸

不要用：

- 过长 Loading
- 过度视差
- 滚动劫持太多
- 按钮被动画遮挡
- 用户找不到导航

## 9. 移动端规范

移动端重点不是炫，而是清楚：

- 首屏 CTA 必须露出
- 产品筛选要变成 Bottom Sheet
- 3D 查看器默认高度不要超过 65vh
- 规格表可横向滚动，但推荐改成卡片式
- 询盘按钮固定在底部
