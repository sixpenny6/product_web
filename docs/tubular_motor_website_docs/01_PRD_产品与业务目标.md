# PRD：管状电机官网产品需求文档

## 1. 背景

公司销售管状电机，产品覆盖遮阳窗帘、小型卷帘门、车库门、大型卷帘门和伺服控制场景。客户在采购前通常关心：

- 产品适配什么场景；
- 电机扭矩、速度、管径、电压、防护等级；
- 是否支持遥控、智能控制、有线控制、限位设置；
- 安装方式、接线方式、附件兼容；
- 是否有说明书、规格书、认证文件；
- 是否能获取报价或技术支持。

官网的核心任务是把这些信息以高级、清楚、可信的方式呈现出来。

## 2. 目标用户

### 2.1 工程商 / 系统集成商

他们需要快速确认产品是否适配工程项目，重点看参数、安装、认证、供货能力。

### 2.2 门窗/遮阳/卷帘门制造商

他们关注批量采购、产品尺寸、适配性、稳定性、OEM/ODM 能力。

### 2.3 经销商

他们关注产品系列完整度、卖点、资料包、售后支持和供货周期。

### 2.4 终端客户，次要用户

他们可能会通过搜索进入网站，需要看懂产品用途，但不一定懂技术参数。

## 3. 项目目标

### 3.1 业务目标

1. 提升官网品牌高级感和可信度。
2. 降低客户理解产品的成本。
3. 提高询盘转化率。
4. 帮助销售团队减少重复解释。
5. 建立可长期维护的产品资料中心。
6. 支持后续多语言和海外市场。

### 3.2 产品目标

1. 每个产品详情页都能完整说明产品。
2. 每个系列都能通过 360°/3D 查看结构和安装细节。
3. 用户能通过筛选快速找到适合的产品。
4. 用户能比较不同型号。
5. 用户能下载规格书、安装说明、认证文件。
6. 后台人员能更新产品，不依赖开发。

## 4. 产品范围

### 4.1 MVP 范围

- 首页
- 产品分类
- 产品列表
- 产品详情
- 360°/3D 查看器
- 产品筛选
- 产品对比
- 应用场景
- 下载中心
- 询盘表单
- CMS 后台
- SEO 基础设置
- 部署上线

### 4.2 第二阶段

- 多语言：中文/英文
- 经销商下载专区
- 在线选型工具
- AR 查看
- 案例库
- 视频中心
- 客户账户
- CRM 集成
- 在线客服
- 自动生成 PDF 产品卡

## 5. 核心转化路径

### 路径 A：首页进入产品

首页 Hero → 产品系列卡片 → 产品列表 → 产品详情 → 360°查看 → 参数表 → 询盘

### 路径 B：按应用场景进入

应用场景 → 推荐系列 → 产品详情 → 下载资料 → 询盘

### 路径 C：技术客户直接查参数

产品中心 → 筛选扭矩/电压/管径 → 对比产品 → 下载规格书 → 联系技术支持

### 路径 D：搜索进入

搜索结果 → 产品详情 → 相关型号 → 询盘

## 6. 成功指标

| 指标 | 目标 |
|---|---|
| 产品详情页停留时间 | 增加 |
| 询盘提交率 | 增加 |
| 产品资料下载次数 | 增加 |
| 产品搜索无结果率 | 降低 |
| 移动端跳出率 | 降低 |
| 销售重复解释问题 | 降低 |

## 7. 非目标

MVP 阶段不做完整在线支付电商，不做复杂 ERP，不做复杂经销商价格系统，不做全自动选型报价。先把展示、询盘和资料体系做好。
