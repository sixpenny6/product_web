# 管状电机官网落地方案：执行总览

## 1. 项目定位

这是一个面向 B2B 客户、渠道商、工程商、门窗遮阳系统集成商的工业产品官网。核心不是“炫技”，而是：

1. 让客户快速判断产品是否适用；
2. 让客户能清楚浏览不同系列；
3. 在产品详情页中通过 360°/3D 查看关键结构；
4. 让客户快速下载资料、比较参数、提交询盘；
5. 后台可持续维护产品、规格、资料、案例和 SEO 内容。

参考风格来源：

- DJI Osmo Pocket 3：科技感 Hero、场景化卖点、分段式产品叙事。
- Dyson OnTrac：高级感、配置器/可视化切换、清晰 CTA。
- Razer Blade 16：暗色科技风、强视觉冲击、参数和性能信息突出。
- Dell PowerEdge XR5610：产品详情页中的 3D 360 查看、AR 查看、规格与支持资料结构。

## 2. 产品分类

初始产品线建议分为 4 类：

| 分类 | 对应产品 | 主要用户 | 页面重点 |
|---|---|---|---|
| 遮阳/窗帘小型管状电机 | 小型遮阳帘、窗帘电机 | 窗饰厂、智能家居集成商 | 安静、体积小、控制方式、适配帘布尺寸 |
| 59 系列电机 | 小卷帘门、小型卷帘系统 | 门窗厂、商铺门系统商 | 扭矩、安装尺寸、卷管适配、安全性 |
| 92 系列大型电机 | 车库门、大型卷帘门 | 工程商、门业厂、经销商 | 大扭矩、耐用、防护等级、安装与维护 |
| 92 伺服电机 | 高端车库门/工业门/精准控制场景 | 工程客户、自动化客户 | 精准控制、反馈、软启停、控制系统兼容 |

## 3. 推荐技术栈

为了降低落地难度，建议采用一个全栈 Next.js 项目，而不是前后端完全分离。

### 前端

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- GSAP，可选，用于复杂滚动叙事
- React Three Fiber + drei + three.js
- Zustand，用于局部 UI 状态
- React Hook Form + Zod，用于表单校验

### 后端 / CMS

推荐优先方案：

- Next.js Route Handlers / Server Actions
- Payload CMS
- PostgreSQL
- Prisma，可选。如果 Payload 已经承担主要数据访问，Prisma 可以只用于自定义业务表。
- Cloudflare R2 / AWS S3，用于 3D 模型、PDF、图片、视频等资源。
- Resend / SendGrid / SMTP，用于询盘邮件通知。
- Vercel / Railway / Render / 自建服务器部署。

### 为什么这样选

1. 对 vibe coding 友好，项目边界清楚。
2. 前端、后台、API 可在一个 repo 内完成。
3. CMS 能让非开发人员维护产品资料。
4. 3D 模型和大文件可走对象存储，避免拖慢主应用。
5. MVP 可先上线，再逐步增强 3D、AR、对比器、经销商门户。

## 4. MVP 必须包含的模块

MVP 不要一次做成复杂电商。第一版建议做“B2B 产品展示 + 询盘 + 资料下载 + 360°查看”。

### 前台页面

1. 首页
2. 产品分类页
3. 产品列表页
4. 产品详情页
5. 360°产品展示模块
6. 应用场景页
7. 技术资料下载页
8. 关于我们
9. 联系/询盘页
10. 搜索结果页

### 后台模块

1. 产品管理
2. 产品分类管理
3. 产品规格管理
4. 3D 模型/图片/PDF 管理
5. 应用场景管理
6. 询盘管理
7. SEO 管理
8. 管理员权限

## 5. 建议开发顺序

1. 静态信息架构和 UI 原型
2. 前端页面骨架
3. 产品数据 JSON Mock
4. 产品详情页 + 规格表 + 下载区
5. 360°/3D 查看器 MVP
6. CMS + 数据库
7. 询盘表单 + 邮件通知
8. 搜索、筛选、对比
9. SEO、性能优化、部署
10. 后台内容录入和验收

## 6. 最小上线标准

第一版上线时至少要做到：

- 每个产品有清晰参数表；
- 每个产品有应用场景；
- 每个产品详情页有询盘入口；
- 至少核心产品有 360°旋转或 3D 模型；
- 页面移动端可用；
- 后台能改产品内容；
- 询盘能进入后台并发送邮件；
- 产品页 SEO 标题、描述、Open Graph 图片完整；
- Lighthouse Performance 建议不低于 80，Accessibility 不低于 90。
