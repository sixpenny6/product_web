# Tubular Motor Frontend Skill

Use when working on:

- Homepage
- Product listing
- Product detail pages
- UI components
- Navigation
- Responsive layout

## Requirements

- Follow `docs/tubular_motor_website_docs/03_UI视觉设计规范.md`.
- Use dark industrial technology style.
- Keep CTA buttons clear and visible.
- Use structured product data.
- Do not fabricate product specifications.
- Mobile layout must be usable.
- Do not let decorative animation block user actions.
- Product pages must help users find products, compare details and submit inquiries.

## UI Priorities

1. Clear navigation
2. Clear product categories
3. Clear CTAs
4. Clear product specs
5. Premium but not confusing visual style
6. Mobile usability
